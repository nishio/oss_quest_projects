# 参加・合意形成 / Civic Tech

* **Decidim（日本展開）** — オープンソースの参加型民主主義プラットフォーム。Code for Japanが国内初の公式パートナーで、日本語化・導入支援を継続（導入30件超の実績）。貢献: 日本語ドキュメント／テーマ実装／事例集。 ([プレスリリース・ニュースリリース配信シェアNo.1｜PR TIMES][1])
* **Jibungoto Planet（自分ごとプラネット）** — 個人のカーボンフットプリント可視化アプリ。NIES×Code for JapanによるOSS化。貢献: 翻訳・UI改善・行動提案のデータ拡充。 ([directory.civictech.guide][2])
* **Where Does My Money Go?（spending.jp）** — 「税金はどこへ行った？」自治体版を量産できるOSS。復活プロジェクトも進行。貢献: 各自治体データ整備／COFOG対応／テンプレ刷新。 ([spending.jp][3])
* **FixMyStreet Japan** — 道路穴や不法投棄などを通報し行政と連携。ベースはmySocietyのAGPL OSS。貢献: 日本向けUI／Open311連携／導入自治体の成果可視化。 ([atmarkit.itmedia.co.jp][4])

# オープンデータ / オープンガバメント基盤

* **e-Govデータポータル（旧DATA.GO.JP統合）** — 行政オープンデータの横断検索・全文検索を提供。CKAN API準拠。貢献: 利用ガイドやSDK、データ品質レポート。 ([Impress Watch][5])
* **CKAN（日本の多数ポータルで採用）** — データカタログOSS。data.go.jpの基盤にも採用されてきた。貢献: 日本語拡張・自治体向けプラグイン。 ([GitHub][6])
* **自治体標準オープンデータセット（正式版）** — データ形式・項目の標準化指針。貢献: 変換ツールの改善／標準遵守の検証スクリプト。 ([デジタル庁][7])
* **地方公共団体向けオープンデータパッケージ（OSS）** — カタログ＋ダッシュボードの配布物。貢献: 令和版テンプレ更新／クラウド導入手順。 ([cio.go.jp][8])
* **法令API v2（e-Gov法令検索）** — 2025/03/19リリース。時点指定取得など機能拡充。貢献: SDK・サンプル、テキスト正規化。 ([e-Gov 法令検索][9])
* **NISTEP「国会・府省会議 議事録メタデータ」** — *.go.jp横断の議事録メタデータをGitHub公開。貢献: 最新URL検証／JSONスキーマ整備。 ([nistep.go.jp][10])
* **SmartNews 国会議案DB（オープンデータ）** — 過去20年分の議案データを整理して無償公開。貢献: 差分検出・可視化・API化。 ([ITmedia][11])

# 危機対応・保健

* **東京都 新型コロナ対策サイト（OSS）** — Vue/Nuxt製でMITライセンス。全国に派生。貢献: 派生版のアーカイブ／再利用テンプレ。 ([プレスリリース・ニュースリリース配信シェアNo.1｜PR TIMES][12])
* **sinsai.info（2011 震災危機マッピング）** — Ushahidiベース。OSM日本コミュニティ中心に運用、1か月で9千件超の報告集約。貢献: 記録の再編／教訓ドキュメント化。 ([ushahidi.com][13])

# 地図・交通

* **GSI Maps（地理院地図のソース公開）** — 国土地理院が本体のソースをBSD-2-Clauseで公開。貢献: レイヤ編集／事例集。 ([GitHub][14])
* **GTFS-JP（標準的なバス情報フォーマット）** — 日本向けGTFS拡張と普及。貢献: バリデータ／作成ガイドの改善。 ([gtfs.jp][15])
* **TokyoGTFS** — 首都圏の鉄道・一部バスのGTFS生成ツール。貢献: 事業者拡張／CIでの定期ビルド。 ([GitHub][16])

# 環境・市民科学

* **Safecast** — 放射線・大気質データをCC0で継続公開（AWS公開データにも登録）。貢献: データ同化・可視化・QA。 ([safecast.org][17])
