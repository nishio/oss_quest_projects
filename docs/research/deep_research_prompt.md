# ChatGPT Deep Research用プロンプト：日本のOpen Government事例

以下のテーマについて、Deep Researchを実施してください：

---

## リサーチテーマ

**日本の政府・自治体によるオープンガバメント、Civic Tech、参加型民主主義の取り組み**

特に以下の条件を満たすプロジェクトや取り組みを調査してください：

### 調査対象の条件

1. **オープン性**
   - オープンソースソフトウェア（GitHubなどで公開）
   - オープンデータ（データセットの公開）
   - オープンAPI（機械可読な形式での提供）
   - 市民参加型プラットフォーム

2. **実施主体**
   - 日本の中央省庁（デジタル庁、総務省、国土交通省など）
   - 都道府県・市区町村
   - 政府関連機関（独立行政法人など）
   - 政府と民間の協働プロジェクト

3. **分野**
   - **参加・合意形成**: 住民参加、パブコメ、デジタル民主主義
   - **透明性**: 議会・行政の可視化、情報公開
   - **市民サービス**: 通報システム、相談窓口、申請手続き
   - **危機対応**: 災害情報、感染症対策
   - **データ公開**: 統計、地図、交通、予算・決算

4. **技術的特徴**
   - Webサイト/アプリとして実装されている
   - API提供がある
   - ソースコードが公開されている
   - 他自治体への横展開が可能

### 特に調べてほしい事例

#### すでに収録済み（参考）
- 東京都新型コロナ対策サイト
- Decidim Japan（Code for Japanによる導入支援）
- FixMyStreet Japan
- sinsai.info
- 広聴AI
- いどばた

#### 追加調査が必要な分野
1. **自治体のデジタル参加プラットフォーム**
   - 住民提案、投票、予算決定への参加
   - パブリックコメントシステム
   - 例：デジタル参加のプラットフォーム実証実験

2. **議会・行政の透明化**
   - 議事録の公開・検索システム
   - 議会中継・アーカイブ
   - 予算・決算の可視化
   - 例：自治体の議会DX事例

3. **市民通報・要望システム**
   - 道路の不具合、公園の問題報告
   - 市民からの相談・要望受付
   - 例：各自治体の通報アプリ

4. **データドリブンな政策形成**
   - EBPM（Evidence-Based Policy Making）ツール
   - データ分析プラットフォーム
   - ダッシュボード

5. **デジタルID・認証基盤**
   - マイナンバーカード活用事例
   - デジタルIDを使った市民サービス
   - 例：xIDプラットフォーム

6. **その他の先進事例**
   - Code for Japan関連プロジェクト
   - Civic Tech団体との協働事例
   - 総務省・デジタル庁の実証事業

### 収集してほしい情報（各事例ごと）

各プロジェクトについて、以下の情報を整理してください：

```
1. **名称**: 正式名称
2. **実施主体**: 省庁名・自治体名・組織名
3. **概要**: 1-2行の簡潔な説明
4. **詳細説明**: プロジェクトの目的、特徴、実績（200-300文字）
5. **公開状況**:
   - Webサイト/アプリのURL
   - GitHubリポジトリ（あれば）
   - API仕様（あれば）
   - ドキュメント（あれば）
6. **オープン性の度合い**:
   - オープンソース？（ライセンス）
   - オープンデータ？（形式）
   - 再利用可能性（他自治体への展開事例）
7. **開始時期・現状**: いつ始まったか、現在も稼働中か
8. **参考情報源**: 公式発表、プレスリリース、報道記事のURL
```

### リサーチの進め方

1. **公式情報源を優先**:
   - 各省庁・自治体の公式サイト
   - デジタル庁の事例集
   - 総務省のオープンガバメント関連ページ
   - GitHub Japan/日本の公的機関

2. **キーワード例**:
   - 「オープンガバメント」「Civic Tech」
   - 「住民参加」「デジタル民主主義」
   - 「市民参加プラットフォーム」
   - 「議会DX」「行政DX」
   - 「EBPM」「データ駆動」
   - 「Code for Japan」「シビックテック」

3. **チェックすべきサイト**:
   - デジタル庁: https://www.digital.go.jp/
   - 総務省: https://www.soumu.go.jp/
   - Code for Japan: https://www.code4japan.org/
   - Civic Tech Guide: https://directory.civictech.guide/
   - GitHub: 政府・自治体組織のリポジトリ

4. **注意点**:
   - 単なる情報公開サイトではなく、「市民参加」「透明性向上」「課題解決」の要素があるもの
   - OSS化されていなくても、「オープンAPI」「データ公開」「市民参加」があれば対象
   - 実証実験で終了したものでも、知見が公開されていれば対象
