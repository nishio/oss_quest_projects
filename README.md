# 日本のオープンソースプロジェクト / Japanese Open Source Projects

日本人が主導・貢献するオープンソースプロジェクトのキュレーションカタログです。

[台湾のOpen StarTer Village](https://github.com/ocftw/open-star-ter-village)にインスパイアされたプロジェクトで、最終的にはボードゲーム形式での提供を目指していますが、現在はWebカタログとして公開しています。

## 🌐 Webサイト

[https://nishio.github.io/oss_quest_projects/](https://nishio.github.io/oss_quest_projects/)

## 📋 収録プロジェクト

44のプロジェクトを3つのカテゴリに分類して紹介：

### Open Source（27プロジェクト）
ML/AI、NLP、DevOps、プログラミング言語、クリエイティブツールなど

- **代表例**: Ruby, Optuna, Fluentd, MeCab, Misskey, VRM/UniVRM, VOICEVOX

### Open Data（12プロジェクト）
政府データ、標準規格、地図、交通、環境データなど

- **代表例**: data.go.jp, e-Gov法令API, GTFS-JP, 地理院地図, Safecast, Polimoney

### Open Government（6プロジェクト）
参加型プラットフォーム、市民通報、危機対応など

- **代表例**: Decidim Japan, 広聴AI, いどばた, FixMyStreet Japan, 東京都新型コロナ対策サイト

## 🎯 選定基準

- **日本との関わり**: 日本人の主要開発者、日本企業・大学の主導、活発な日本語コミュニティ
- **検証可能性**: 公式サイト、GitHub、財団発表などの一次情報
- **活動レベル**: 最近の更新（18ヶ月以内）、活発なIssue/PR対応
- **オープン性**: 明確なOSSライセンス、ドキュメント、貢献経路

## 🚀 将来の展望

このプロジェクトは段階的に発展していきます：

1. **現在（フェーズ1）**: Webカタログの充実
2. **フェーズ2**: ゲーム設計（jobs/requirementsの設定）
3. **フェーズ3**: ボードゲーム版の実装・公開
